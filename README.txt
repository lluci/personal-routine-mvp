# Routine MVP

Aquesta és una petita webapp experimental pensada per provar un flux diari de rutina personal. El projecte inclou una sola pàgina (`index.html`) amb una interfície bàsica tipus “Today”: targetes per a cada bloc del dia, un “Morning check-in” i botons placeholders per simular accions futures (Done, Snooze, Add reason).

L’objectiu és avaluar la usabilitat, el ritme d’interacció i la sensació d’ús abans de connectar cap base de dades o backend real. La pàgina funciona com un **static site** i es pot desplegar fàcilment amb **GitHub Pages**.
